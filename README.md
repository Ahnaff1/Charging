# Charging
1st Step :
      Clone the repo
 
2nd Step:
       Do npm install


3rd step :
        Add Gmap api key in android/app/src/main/AndroidMainifest.xml
       /***example**/
        <meta-data
        android:name="com.google.android.geo.API_KEY"
        android:value="*******YOUR API KEY*****"/>

Now you can build and run the project
